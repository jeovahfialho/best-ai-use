# CLAUDE.md - Best AI Use

## Sobre o Projeto
Repositorio central de boas praticas para desenvolvimento assistido por IA.
Serve como base de conhecimento referenciavel via CLAUDE.md, AGENTS.md ou GEMINI.md em outros projetos.

## Estrutura
- `claude/` - Boas praticas com Claude (Code, API, prompting, hooks, MCP)
- `codex/` - Boas praticas com OpenAI Codex (CLI, API, workflows)
- `gemini/` - Boas praticas com Google Gemini (CLI, API, IDEs)
- `tools/` - Ferramentas complementares (Context Mesh, Context7, Promptfoo)

## Convencoes
- Conteudo em portugues brasileiro
- Foco em praticidade e aplicabilidade
- Cada guia inclui fontes e referencias
- Novos artigos e ferramentas sao adicionados continuamente

## Como Usar em Outros Projetos
Referencie os guias deste repositorio no CLAUDE.md de outros projetos para que agentes de IA sigam as boas praticas documentadas aqui.
